# smart-pension-service

## 项目介绍

1. 名称：智慧养老服务平台
2. 主要功能
   - 登录注册
   - 检索养老服务
   - 热门推荐

## 技术架构

1. 前端基于 vue 及 vue-router、vuex、组件库 iview ui
2. 使用 Node 转发请求，访问 Java api 接口
3. 后端采用 Java Web SSM 框架

## 开发环境

1. vue：vue-cli 3
2. node：Node.js 12.5
3. java：jdk8
4. database：mySql 5.7
5. tool：idea + vs code
